# Custom Data Structures Library (C++)

A custom implementation of fundamental data structures built from scratch in C++ without using STL containers.

## Implemented Data Structures

### Stack
- Push
- Pop
- Peek
- isEmpty()

### Queue
- Enqueue
- Dequeue
- Peek
- isEmpty()

### Linked List
- Insert at end
- Delete node
- Display traversal

### Binary Search Tree
- Insert nodes
- Inorder traversal

---

## Concepts Practiced

- Object-Oriented Programming
- Arrays
- Dynamic Memory Allocation
- Pointers
- Recursion
- Tree Traversal
- Data Structure Design
